📊 Credit Card Fraud Detection (Synthetic Dataset)
SQL + Python + Machine Learning Project

This project simulates a real financial data pipeline used to analyze and detect credit card fraud across multiple countries in the Americas.
It combines synthetic data generation, SQL modeling, Python analytics, and machine learning to demonstrate end-to-end data analysis and fraud detection capabilities.
